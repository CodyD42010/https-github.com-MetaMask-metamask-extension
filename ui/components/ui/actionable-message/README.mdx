# Actionable Message

Popup component that give the user information. Actionable Message component can generate a tooltip and a maximum of two action buttons.

## Usage

The following describes the props and example usage for this component.

| Prop Type         | Description                                                         |
| ----------------- | ------------------------------------------------------------------- |
| `message`         | text inside actionable message                                      |
| `primaryAction`   | first button props that have label and onClick props                |
| `secondaryAction` | second button props that have label and onClick props               |
| `className`       | add css classname for the component based on the parent css         |
| `type`            | change color theme for the component already predefined in css |
| `withRightButton` | change text align to left and button to bottom right                |
| `infoTooltipText` | add tooltip and custom message                                      |
| `useIcon`         | add tooltip icon in the left component without message              |
| `iconFillColor`   | change tooltip icon color                                           |
