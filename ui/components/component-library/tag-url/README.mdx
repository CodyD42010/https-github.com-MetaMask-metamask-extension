# Tag

The `TagUrl` is a component used to display Dapp information within a container.

## Props

The `Tag` accepts all props below as well as all [Box](/docs/ui-components-ui-box-box-stories-js--default-story#props) component props.

<ArgsTable of={TagUrl} />

### Image Source

Use the `imageSource` prop with an image url to render the  `AvatarFavicon`.

### Label

Use the `label` prop for the text content of the `TagUrl` component

### Cta

If we want to have a button in `TagUrl`, cta is an object that contains the `label` and callback of the `Button`.
