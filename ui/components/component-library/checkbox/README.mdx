# Checkbox

Checkbox is a graphical element that allows users to select one or more options from a set of choices.

<Canvas>
  <Story id="components-componentlibrary-checkbox--default-story" />
</Canvas>

## Props

The `Checkbox` accepts all props below as well as all [Box](/docs/components-ui-box--default-story#props) component props

<ArgsTable of={Checkbox} />

### Label

Use the `label` string prop to set the label of the `Checkbox`

<Canvas>
  <Story id="components-componentlibrary-checkbox--label" />
</Canvas>

```jsx
import { Checkbox } from '../../component-library';

<Checkbox label="Checkbox Label" />;
```

### LabelProps

Use the `labelProps` object prop to change the label `Label` component

<Canvas>
  <Story id="components-componentlibrary-checkbox--text-props" />
</Canvas>

```jsx
import { Checkbox } from '../../component-library';
import {
  TextColor,
  FontWeight,
} from '../../../helpers/constants/design-system';

<Checkbox
  label="Checkbox Label with labelProps"
  labelProps={{ color: TextColor.errorDefault, fontWeight: FontWeight.Bold }}
/>;
```

### OnChange

Use the `onChange` function prop to set the onChange handler of the `Checkbox`

<Canvas>
  <Story id="components-componentlibrary-checkbox--on-change" />
</Canvas>

```jsx
import { Checkbox } from '../../component-library';

const [isChecked, setIsChecked] = React.useState(false);

<Checkbox onChange={() => setIsChecked(!isChecked)} isChecked={isChecked} />;
```
