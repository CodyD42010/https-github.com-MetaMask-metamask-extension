# ButtonBase

The `ButtonBase` is the base component for all icons. It is used in conjunction with a script to create all icons it should not be used directly.

### Size

Use the `size` prop and the `SIZES` object from `./ui/helpers/constants/design-system.js` to change the size of `ButtonBase`. Defaults to `SIZES.SM`

Possible sizes include:

- `SIZES.XS` 22px
- `SIZES.SM` 32px
- `SIZES.MD` 40px
- `SIZES.LG` 48px
