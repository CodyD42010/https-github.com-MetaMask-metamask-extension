# AvatarToken

The `AvatarToken` is a component responsible for display of the image of a given token

## Props

The `AvatarToken` accepts all props below as well as all [BaseAvatar](/ui-components-component-library-base-avatar-base-avatar-stories-js--default-story) component props.

<ArgsTable of={AvatarToken} />

###  tokenName

Use the ` tokenName` prop to set the initial alphabet of the `AvatarToken`.

###  tokenImageUrl

Use the `tokenImageUrl` prop to set the image to be rendered of the `AvatarToken`.
