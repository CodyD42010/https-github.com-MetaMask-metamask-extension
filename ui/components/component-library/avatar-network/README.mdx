# AvatarNetwork

The ` AvatarNetwork` is a component responsible for display of the image of a given network

## Props

The `AvatarNetwork` accepts all props below as well as all [BaseAvatar](/ui-components-component-library-base-avatar-base-avatar-stories-js--default-story) component props.

<ArgsTable of={AvatarNetwork} />

###  networkName

Use the ` networkName` prop to set the initial alphabet of the `AvatarNetwork`.

###  networkImageUrl

Use the `networkImageUrl` prop to set the image to be rendered of the `AvatarNetwork`.
